This is Team 2's project, which portrays the workings of a photo studio. 
Made over the course of the semester.

FEATURES:
1. Three distinct modes:
Receptionist - checks the current situation of the photo studio. How much revenue has been made, how many resources are left, order count etc.
Customer - user can place orders through this mode, input their name, how many photos they'd like to print/develop etc.
Photographer - a simple mode that takes on orders and completes the necessary amount of photos that are needed.
2. Dynamic arrays.
3. Save/load system.
